Large File Splitter to help handle large Excel and CSV files.
 
Features:
Supports .xlsx and .csv files
Splits large datasets into multiple worksheets based on a configurable row limit
Converts CSV files to Excel format (automatic delimiter detection)
No Python installation required; simply run the executable
This tool may be useful when working with files that exceed Excel's worksheet limits or are too large to manage conveniently. It can also be used as a quick CSV-to-Excel (.xlsx) converter.
